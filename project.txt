# Personal OS - Project System Specification V1

## Purpose

The Project System tracks all personal, academic, and professional projects.

Projects are the practical output of skills.

Skills answer:

"What am I learning?"

Projects answer:

"What am I building?"

---

→ Planned
→ Building
→ Testing
→ Completed

## Scheduling

* Days Active
* Progress
* Last Activity

---

### Deadline

Requires:

System calculates:

* Days Remaining
* Expected Progress
* On Track
* Behind Schedule

## Progress Formula

Completed Tasks ÷ Total Tasks

Example:

8 completed

10 total

Progress = 80%

# Automation Rules

When created:

Created At = Current Date

When status becomes Building:

Started At = Current Date

When status becomes Completed:

Completed At = Current Date

Progress = 100%

When modified:

Updated At = Current Date

When all checklist items complete:

Suggest completion

Do not automatically mark completed.

User decides.
