# Personal OS - System Specification V1

## Purpose

Personal OS is a personal productivity and life-tracking system.

It is NOT:

* A Notion clone
* A tracker builder
* A database builder
* A multi-user application

It is designed specifically for personal use and focuses on:

* Skills
* Projects
* Nutrition
* Fitness
* Goals
* Learning Progress

The primary objective is to provide a clear overview of personal growth, learning, health, and project execution.

---

# Core Principles

1. Fast data entry
2. Minimal friction
3. Automatic date tracking
4. Progress should be calculated automatically wherever possible
5. Dashboard-first design
6. Long-term trend tracking
7. Simple over flexible
8. Purpose-built, not generic

---

### Automatic Date Rules

When created:

Created At = current date

When moved to Active:

Started At = current date

When moved to Completed:

Completed At = current date

Whenever edited:

Updated At = current date

Dates should never require manual entry.

---

### Skill Checklist System

Every skill can contain checklist sections.

Example:

Playwright

Fundamentals

* Install Playwright
* Launch Browser
* Open Page
* Locate Elements

Interactions

* Click Elements
* Fill Forms
* Upload Files

Projects

* Build Scraper
* Build Automation Tool

---

Progress Calculation

Progress should be automatically calculated:

Completed Items / Total Items

Example:

8 completed

10 total

Progress = 80%

No manual percentage entry required.
